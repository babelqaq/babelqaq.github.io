发表于 2024-03-02 更新于 2024-03-03 Valine： 本文字数： 6.2k 阅读时长 ≈ 11 分钟

规则总览

本挑战具有十个固定设定的人物，每个人物需达成指定挑战，全部达成则挑战完成

人物寿命需设置为普通

初始房子推荐选择较大地块，初始具有一个孤儿院建筑（所有设备尽量精简低级），除Adam之外，其他孩子都使用集体宿舍及上下铺（可以男女分开或者不分），家庭初始资金为1000模拟币。

所有人物（除Adam之外）找到相应工作且完成要求后需搬出原始家庭成为与原始家庭同一地段的租户。

必须接受所有游戏过程中自动出现的偏好，性格转变及生活方式。

玩家只能操纵Adam与Charles完成家务。

玩家可以进行Polaris，Jasmine，Michael，Cacao奇想中的友情/浪漫交互，Ocrea可任意与非家庭中的异性进行友情/浪漫交互，Adam可任意与其他人物进行亲情交互。除此之外，玩家不得操控其他人物进行友情/浪漫交互。

每个节日假期必须出门旅行

Ocrea接受所有外出邀请，Jasmine接受所有异性外出邀请，Adam接受所有来自家人的外出邀请，Raven拒绝所有外出邀请，其他外出邀请则通过随机数决定。

Polaris与Sirius应设置为姐弟关系，Jasmine与Micheal应设置为姐弟关系

人物规则

1 Adam（吴铮）

初始年龄：青年

初始性格特征：秋之子，好人，乐于助人的

要求：

  1. 赚钱方式只能选择写作/绘画，且不得外出工作
  2. 完成畅销作家的全部抱负内容并完成一本人生之书
  3. 对所有儿童都采取鼓励式教育，冷静/坚定地告诫（不得使用严厉的告诫），以及拥抱和家人亲吻
  4. 达成写作/绘画十级
  5. 在每位家人离开家前为他们绘制一副肖像画
  6. 在所有孩子离开家前与他们成为好朋友

2 Charles（齐彭殇）

初始年龄：青少年

初始性格特征：洁癖，有能力的（青少年）

要求：

  1. 完成所有作业及加分作业，以A类学生从高中毕业
  2. 在保证A类学生的前提下在学校尽可能多地交朋友
  3. 具有一份青少年兼职并从不缺席
  4. 获得政治学大学学位并完成学术的全部抱负
  5. 具有负责任的奖励特征

职业：政治家

3 Polaris（张紫薇）

初始年龄：青少年

初始性格特征：天才，夏之子（青少年）

要求：

  1. 将灵魂伴侣作为终身抱负
  2. 若与Charles具有浪漫关系，则与Charles约会至少一次
  3. 逻辑五级及以上

职业：特工

4 Ocrea（高秋）

  2. 魅力五级及以上
  3. 每次上学都选择交朋友
  4. 加入一个俱乐部并与俱乐部里每位异性发展浪漫关系
  5. 去至少两次夜店

  3. 至少交五个异性朋友
  4. 若有异性提出交往/求婚，则立刻同意

职业：

若与他人发展出浪漫关系，则不选择职业，成年后立刻与其结婚，抱负改为欢乐大家庭

若无浪漫关系，则选择职业为主厨

初始性格特征：粘人精（幼儿）社交恐惧症（儿童）

  2. 在离开家前可以与其他人发展浪漫关系，但不与任何人确认成为恋人
  3. 和一个玩偶熊/鬼魂成为朋友
  4. 在网络上拥有至少一千位追随者
  5. 魅力及健身三级及以上
  6. 成为二星名人及以上
